=======
Bookman
=======

Bookclub Manager

Developers
==========
`Pipenv <https://docs.pipenv.org/en/latest/>`_ required.

Install Dependencies
--------------------
.. code-block::

    pipenv sync --dev

Run Tests
---------
.. code-block::

    pipenv run test

Generate Documentation
----------------------
From the project root::

    pipenv run doc

Start Documentation Web Server
------------------------------
From the project root::

    pipenv run hostdoc
